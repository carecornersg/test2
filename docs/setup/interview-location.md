# Interview Location Setup

This guide explains how to manage interview locations used in CAS picklists.

## Overview

Interview locations are used when creating a case (**Interview Schedule → Location**). Keeping this list accurate helps Call Centre Staff select the correct venue quickly.

---

## Create Interview Locations

### Step 1: Go to Interview Location

In the left-hand side menu, navigate to **Interview Location**.

![Create Interview Location 1](../images/Create%20interview%20location%201.png){: style="max-width:850px" }

### Step 2: Click Create New Interview Location

Fill in the following fields:

- **Active** – Toggle on/off to activate or deactivate the location
- **Interview Location Name** – Location name displayed in the case picklist
- **Address** – Full address
- **Block/Building** – Block or building name (if applicable)
- **Postal Code** – Postal code
- **Region** – Select the region from the dropdown

![Create Interview Location 2](../images/Create%20interview%20location%202.png){: style="max-width:500px" }

### Step 3: Click Create

Click **Create** to create a new interview location.

---

## Edit Interview Location

### Step 1: Find the Location

Go to **Interview Location** and find the location you want to update

### Step 2: Click Edit

Click the **Edit** (pen) icon.

![Edit Interview Location 1](../images/Edit%20interview%20location%201.png){: style="max-width:850px" }

### Step 3: Update Details

Update fields as needed:

- Edit **Interview Location Name**, **Address**, **Block/Building**, **Postal Code**, or **Region**
- Toggle **Active** off to inactivate the location

![Edit Interview Location 2](../images/Edit%20interview%20location%202.png){: style="max-width:500px" }

### Step 4: Save Changes

Click **Update** to save changes.
