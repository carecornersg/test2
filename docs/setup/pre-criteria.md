# Pre-criteria

This guide outlines the prerequisites before using CAS.

## Overview

Before using the CAS system, ensure you have the following requirements ready.

---

## Quick Start

Before you begin:

1. Make sure you have valid login credentials for the **CAS Web Portal** (Call Centre / Manager / System Administrator)
2. Confirm your **role** is correctly assigned

## Requirements

### 1. Valid Login Credentials

Ensure you have valid login credentials for CAS Web Portal.

- Contact your System Administrator if you don't have credentials
- First-time users will receive a link to setup password

### 2. Access Role

Ensure you have the appropriate role assigned:

- **Call Centre Staff** (Standard): Create cases, monitor broadcasts, view basic operational reports
- **AAYS Manager** (Full): All Call Centre functions + full reports + AA management
- **System Administrator** (Admin): All functions + system configuration + user management
