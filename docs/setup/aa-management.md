# AA Management

This guide explains how AAYS Managers and System Administrators manage Appropriate Adult (AA) profiles and availability in the CAS Web Portal.

## Overview

AA data is primarily maintained in **Salesforce VPRMS** and synced to CAS. In the Web Portal, managers typically use AA management tools to:

- Search and view AA profiles
- Override availability (when needed for operations)
- Review qualifications and preferences that affect broadcast eligibility

---

## Quick Start

1. Open **AA Management**
2. Search/filter for an AA
3. Review profile details (qualifications, regions, timing preferences, status)
4. Apply overrides if required (availability / pause / notes)

---

## Manage AAs

### Step 1: Open AA Management

In the left-hand side menu, navigate to **AA Management**.

### Step 2: Find an AA

- Use search and filters to find the AA you need (e.g., by name, email)
- Click the **View** (eye) icon to open the AA profile

![AA Management](../images/AA%20management%201.png){: style="max-width:800px" }

### Step 3: Review Profile

Common information includes:

- **AA identity**
- **Qualifications**
- **Regions**
- **Timing preferences**
- **Availability / pause status**
- **AA Statistics**

![AA Profile Details](../images/AA%20Profile%20detail.png){: style="max-width:800px" }

### Step 4: Edit AA Profile

**✏️ Who can edit  AA profiles:** Only **AAYS Manager** and **System Administrator** can edit AA profiles.

If you are **Call Centre Staff**, the edit pen icon will appear **greyed out**, meaning the profile is **read-only** in CAS.

Depending on permissions and configuration, editable fields typically include:

- **Contact Number**
- **Address**
- **Primary Region** and **Secondary Region**
- **Preference Time Slots**
- **Qualifications**
- **Low Priority** (toggle)
- **Notification Preferences** (in the Notification Preferences tab, if available)

![Edit AA Profile 1](../images/Edit%20AA%20profile%201.png){: style="max-width:700px" }

![Edit AA Profile 2](../images/Edit%20AA%20profile%202.png){: style="max-width:600px" }

### Step 5: Save Changes

Click **Save** to apply changes.
