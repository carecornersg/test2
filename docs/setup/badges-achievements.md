# Badges & Achievements

This guide explains how administrators/AAYS Manager configure badge/achievement features in CAS.

## Overview

Badges and achievements are designed to encourage AA engagement and recognise contributions. AAs typically view badges in the **CAS Mobile App**.

---

## Configure Badges

### Step 1: Navigate to Achievements

In the left-hand side menu, navigate to **Achievements**

![Create Badge 1](../images/Create%20badge%201.png){: style="max-width:800px" }

### Step 2: Define Badge Rules

Click **Create New Badge** and fill in details:

- **Badge name**
- **Description**
- **Rule Type** (e.g., documentation compliance, punctuality, special recognition)
- **Thresholds** (what counts as “earned”)
- **Badge image**

![Create Badge 2](../images/Create%20badge%202.png){: style="max-width:600px" }

### Step 3: Save and Verify

Click **Create**, then confirm:

- AAs can see the badge menu (if enabled)
- Badge progress and earned notifications behave as expected

---

## Edit Badge

### Step 1: Select a Badge

Go to **Achievements** and select the badge you want to edit.

### Step 2: Click Edit

Click the **Edit** (pen) icon.

![Edit Badge 1](../images/Edit%20badge%201.png){: style="max-width:800px" }

### Step 3: Update Badge Details

Update fields as needed:

- **Badge name**
- **Description**
- **Rule Type**
- **Thresholds**
- **Badge image**

![Edit Badge 2](../images/Edit%20badge%202.png){: style="max-width:600px" }

### Step 4: Save Changes

Click **Update** to apply changes.

---

## View AAs Who Earned a Badge

### Step 1: Open Badge Details

Go to **Achievements** and open the badge you want to review.

![Earn Badge 1](../images/Earn%20badge%201.png){: style="max-width:800px" }

### Step 2: View Earned List

View the list of AAs who have earned the badge.

![Earn Badge 2](../images/Earn%20badge%202.png){: style="max-width:700px" }

---

## Notes

- Badge definitions and thresholds are typically policy-driven (program rules).
- If gamification is not enabled, the menu may be hidden on the AA Mobile App.
