    üìù Create Case

    This guide will walk you through creating a new case in CAS.

    Overview

    When an Investigation Officer (IO) calls to request an Appropriate Adult, you will create a new case in CAS. The system will then broadcast to eligible AAs based on the case requirements.

    Quick Start (Most common flow)

    1. Go to Cases ‚Üí Create New Case
    2. Add youth information and number of AAs required
    3. Set interview schedule and location
    4. Add IO contact details
    5. Add special requirements (optional)
    6. Choose Assignment Method (Broadcast, manual assignment or data entry)
    7. Click Start Broadcast

    How Case Broadcast Works

    Broadcast Types:

    Type Name AA Pool When Used
    B1 Standard Broadcast Primary region AAs First broadcast
    B2 Secondary Region Primary + Secondary region AAs Re-broadcasts
    B3 Staff-Only AA Staff only (ignores preferences) Urgent final attempt

    Broadcast Flow:

    1. Broadcast 1 (B1) ‚Üí Case sent to AAs in primary region
    2. Holding Period (5 mins) ‚Üí System collects multiple responses
    3. Weighted Selection ‚Üí If multiple responses, system selects based on workload:
    4. Bucket A (0-2 cases/month): 60% chance
    5. Bucket B (3-6 cases/month): 30% chance
    6. Bucket C (7+ cases/month): 10% chance
    7. Waiting Period (10 mins) ‚Üí FCFS if no response during holding
    8. Confirmation (5 mins) ‚Üí Selected AA must confirm assignment
    9. Broadcast 2 (B2) ‚Üí Expands to secondary regions if no confirmation
    10. Broadcast 3 (B3) ‚Üí Staff-only broadcast for urgent cases

    Urgency Classification:

    Classification Condition Broadcast Sequence
    Standard Interview > 4 hours away B1 ‚Üí B2 ‚Üí B2...
    Urgent Interview ‚â§ 4 hours away B1 ‚Üí B2 ‚Üí B3
    Last Minute Interview ‚â§ 90 mins (AA cancels) Force B3 immediately

    8-Stage Filtering: Before broadcast, system filters AAs by: Qualifications ‚Üí Region ‚Üí Timing ‚Üí Gender ‚Üí Language ‚Üí Race ‚Üí Cooldown ‚Üí Quiet Hours

    Create New Case

    Step 1: Navigate to Cases

    Login to CAS Web Portal. In the left-hand side menu, navigate to Cases

    Step 2: Click Create New Case

    Click Create New Case button

    Step 3: Fill Youth Information

    Fill in the youth details:

    • Youth Name ‚Äì Full name of young suspect (required)
    • Youth Age ‚Äì Age of youth (6-17 years)
    • Youth Gender ‚Äì Male / Female / Other (required)

    If more than one youth, click Add New Youth to add additional youth (maximum 7).

    Then input the Number of AAs Required for this case.

    Step 4: Fill Offense Details

    Select the Specific Offense from the offense list dropdown.

    Step 5: Specify Interview Schedule

    • Interview Date ‚Äì Date of interview
    • Interview Time ‚Äì Time of interview
    • Location ‚Äì Police station or other location
    • House Visit ‚Äì Check if interview requires house visit

    If House Visit is checked, the House Visit qualification requirement will be automatically added.

    Step 6: Fill IO Information

    • IO Name ‚Äì Investigation Officer's full name (required)
    • Contact Number ‚Äì Primary phone (8 digits, starts with 6/8/9)
    • Alt Contact Number ‚Äì Alternative phone, optional (8 digits, starts with 6/8/9)

    Step 7: Select Special Requirements (Optional)

    If the case requires specific AA characteristics:

    • Gender ‚Äì Male or Female AA preferred
    • Race ‚Äì Preferred ethnicity
    • Language ‚Äì Required spoken language
    • Qualifications ‚Äì Additional qualifications needed

    Step 8: Choose Assignment Method

    • Broadcast ‚Äì Notify all eligible AAs via automated selection.
    • Manual Assignment ‚Äì Select a specific AA directly.
    • Data Entry Only ‚Äì Record completed case without notification. Use for cases completed offline, for records only.

    Broadcast:

    If you choose Broadcast, CAS will show:

    • The number of eligible AAs for this case (based on the criteria you entered)
    • Case-level assignment timing parameters that you can adjust for this specific case before starting:
    • Holding Period
    • Waiting Period
    • Confirmation Period

    Manual Assignment:

    If you choose Manual Assignment, you can view the list of eligible AAs, pick an AA by name and select manual assignment reason droplist.

    Urgent manual assignment: If the manual assignment reason is Urgent, the AA must respond within 3 minutes.

    Data Entry Only:

    This method is used to record cases that were completed offline ‚Äî no notification will be sent to AA, and the case goes directly to Completed status.

    • AA Name ‚Äì Shows all AAs for selection (search by name available)
    • Youth Name ‚Äì Auto-filled from youth info entered above

    Step 9: Start Broadcast/ Assignement or Save

    • Start Broadcast or Start Assignment ‚Äì Save and start broadcast/ assignment immediately
    • Save as Draft ‚Äì Save case to broadcast/ assignment later
